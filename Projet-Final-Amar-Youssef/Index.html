Bienvenue


Si vous êtes ici c'est que vous vous intéressez de près ou de loin aux technologies de vision thermique et infrarouge. Le but de ce site est d'apporter des informations vulgarisé concernant le fonctionnement et les applications de ce type de technologie. Vous trouverez ci-dessous les différentes sections détaillées et accessibles sur ce site.


Attention


Il est important de noté que les informations apportées ici ne sont pas extrêmement avancées, les prérequis nécessaire pour comprendre les sciences de l'infrarouge sont bien trop spécifique pour la plupart d'entre vous (moi y compris) c'est pour cela qu'il a été précisé plus haut que ce site est une vulgarisation de ces informations, c'est-à-dire, une simplification. Cependant si vous avez la curiosité ou le niveau d'étude pour consulter des informations plus avancé que grand bien vous fasse.
